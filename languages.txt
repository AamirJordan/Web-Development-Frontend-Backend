------------------------------------------------------------	Angular 8   -----------------------------------------------------------------------------------------------------------


				***********			ALL ANGULAR PROGRAMS ARE DONE ON NODEJS COMMAND PROMT			*************



Installing angular on your PC	----	1. Open node.js command prompt.
					2. Install "npm install -g @angular/cli" under C:/Users/Admin.
					3. To create a project type "ng new name".

Run a program			----	1. Open the directory I mean the app name for example "C:/Users/Admin/Atom/Angular8/first_app"
					2. Then in the app directory i.e "C:/Users/Admin/Atom/Angular8/first_app" type "ng -open serve" which will run the server.


Create a SubServer		----	in the app directory i.e "C:/Users/Admin/Atom/Angular8/first_app" type "ng generate component SubServer" which will create the SubServer.



Installing bootstrap on a project	----	Within the directory i.e "C:/Users/Admin/Atom/Angular8/first_app" type "npm install --save bootstrap@3"
